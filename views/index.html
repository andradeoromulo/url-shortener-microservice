API Project: URL Shortener Microservice

This is my project of the URL Shortener Microservice challenge for the freeCodeCamp API and Microservice certification. It was built based on the boilerplate available here.

Create short link:

If you want to create a short link, fill the input below with the url:

URL Shortener

You'll receive an object like this one as output:

{"original_url":"https://freecodecamp.org","short_url":"1904412960"}

Access short link:

To access a short link and be redirected to the original URL, just hit this endpoint:

/api/shorturl/{short_url}
